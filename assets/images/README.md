# Portfolio Images

This folder contains all images for the portfolio website.

## Required Images:

1. **profile.jpg** - Your profile picture (recommended size: 400x400px)
   - Square format
   - Professional headshot
   - Place in: `assets/images/profile.jpg`

2. **Portfolio Images** - Project screenshots (recommended size: 800x600px)
   - portfolio-01.jpg
   - portfolio-02.jpg
   - portfolio-03.jpg
   - portfolio-04.jpg
   - portfolio-05.jpg
   - portfolio-06.jpg
   - Place in: `assets/images/`

## Image Guidelines:

- **Profile Picture**: Use a high-quality, professional photo with good lighting
- **Portfolio Items**: Use screenshots or images that best represent your work
- **Format**: JPG or PNG
- **Optimization**: Compress images to reduce file size while maintaining quality
- **Alt Text**: Update the alt text in index.html for accessibility

## Temporary Placeholders:

You can use placeholder image services temporarily:
- https://via.placeholder.com/400x400 (for profile)
- https://via.placeholder.com/800x600 (for portfolio items)

## Adding Your Images:

1. Save your images in this folder
2. Name them according to the list above
3. Update the HTML if you want to use different filenames
4. Ensure images are optimized for web use
